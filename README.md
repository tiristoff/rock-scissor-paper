# rock-scissor-paper

The back is based on Spring boot and front end on React builded with Next.js framework

## Required Software

Java 11
Maven
Npm

## Software:

Java 11
Spring boot
Maven
H2
React 
Next.JS

#How to run the project

mvn clean install spring-boot:run 

cd front

npm run dev
